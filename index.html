About Hash Algorithms

MD5

MD5 (128-bit)

Legacy algorithm, included for reference and backward compatibility. Not recommended for security purposes.

SHA-1

SHA-1 (160-bit)

Deprecated but still referenced in older systems. Included for legacy support.

SHA-256

SHA-256 (256-bit)

Recommended for electronic evidence. Part of SHA-2 family, widely used in forensics and legal contexts.

SHA-512

SHA-512 (512-bit)

Highest security in SHA-2 family. Provides 512-bit output for maximum collision resistance.

SHA3-256

SHA3-256 (256-bit)

Latest NIST standard. Uses Keccak algorithm, provides additional security assurance.

BLAKE2b

BLAKE2b-256 (256-bit)

Modern, high-performance algorithm. Faster than SHA-256 while maintaining security.
