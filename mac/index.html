Important: MAC Address Randomization

  • Modern smartphones (iOS 14+, Android 10+) use randomized MAC addresses by default when scanning for networks
  • A device may show different MAC addresses on different networks
  • The real MAC address is typically only visible when actually connected to a network
  • Random MACs usually have the locally administered bit set (second character is 2, 6, A, or E)
  • This feature was introduced for privacy protection and is now standard behavior

About MAC Addresses

What is MAC Address?

Media Access Control (MAC) address is a unique 48-bit identifier assigned to network interface controllers. It's used for communication within a network segment.

OUI (Vendor Prefix)

The first 3 bytes (6 characters) identify the manufacturer through the Organizationally Unique Identifier (OUI) registered with IEEE.

MAC Randomization

Modern devices randomize MAC addresses when scanning WiFi networks for privacy. This prevents tracking users across locations.
